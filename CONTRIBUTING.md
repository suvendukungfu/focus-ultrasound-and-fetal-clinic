# Contributing to Focus Ultrasound

First off, thank you for considering contributing to Focus Ultrasound! It's people like you who make this project better for everyone.

## Our Tech Stack

- React + TypeScript
- Tailwind CSS + Shadcn UI
- Vite

## Development Process

1. Fork the repo and create your branch from `main`.
2. Install dependencies: `npm install`.
3. Run the development server: `npm run dev`.
4. If you've added code that should be tested, add tests.
5. Ensure the test suite passes: `npm run test`.
6. Format your code with Prettier/ESLint.

## Pull Request Guidelines

- Use descriptive titles for your PRs.
- Link the PR to a specific issue if applicable.
- Provide screenshots for UI changes.
- Ensure CI checks pass before requesting a review.

## Code Style

- Use functional components and hooks.
- Follow the existing folder structure.
- Use descriptive naming constants and variables.
- Write meaningful comments for complex logic.

## Questions?

Feel free to open an issue or contact the maintainers.
