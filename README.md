### Prerequisites

- Node.js (v18.0.0 or higher)
- npm (v9.0.0 or higher)

### Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/your-username/focus-ultrasound.git
   cd focus-ultrasound
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Set up environment:**

   ```bash
   cp .env.example .env
   ```

4. **Launch development server:**
   ```bash
   npm run dev
   ```
